# Política de Privacidade — SaúdeTracker

*Última atualização: 07 de julho de 2026*

O SaúdeTracker é um aplicativo pessoal de acompanhamento de saúde e treino, de uso individual.

## Dados coletados

O aplicativo armazena, mediante autorização explícita do usuário:

- Dados de check-in diário: variabilidade da frequência cardíaca (HRV), frequência cardíaca de repouso, horas e qualidade de sono, humor e notas pessoais.
- Dados de treino: atividades físicas, duração, frequência cardíaca e carga de treino, incluindo dados importados do Strava e do WHOOP mediante conexão OAuth autorizada pelo usuário.

## Uso dos dados

Os dados são utilizados exclusivamente para exibir métricas, tendências e recomendações de treino ao próprio usuário e ao seu treinador autorizado dentro do aplicativo. Nenhum dado é vendido, compartilhado com terceiros ou utilizado para publicidade.

## Integrações (Strava, WHOOP)

As conexões com serviços de terceiros usam OAuth 2.0. Os tokens de acesso são armazenados de forma segura e usados apenas para importar os dados descritos acima. O usuário pode desconectar qualquer integração a qualquer momento na tela de Configurações, o que interrompe a importação de dados.

## Armazenamento e exclusão

Os dados são armazenados no Supabase com controle de acesso por usuário (Row Level Security). O usuário pode solicitar a exclusão completa de seus dados a qualquer momento pelo contato abaixo.
